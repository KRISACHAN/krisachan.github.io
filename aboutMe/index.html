            # 『Hello CSS』系列

            『Hello CSS』是我的处女作，是我第一次写文章的尝试。

            其实写这个系列的想法萌芽于两年前，一次偶然的机会知道了信息无障碍这个概念。

            了解了这个概念之后，我心很受触动。

            因为自己有近视眼，即使带了眼睛，但是一到晚上还是会看不清楚，心很慌，想到自己只是近视，已经如此的不便，那么我们的盲人朋友或者是行动不便的朋友又会是如何呢？

            之后就一直想为有网页浏览障碍的朋友做些什么，但是当时由于各种原因，只能把这个想法深藏于心里。

            一直到了去年，去参加了一场前端分享会，里面一节分享是由我们有视力障碍的朋友做的，听完分享之后，久久不能自已，遂有这个系列的诞生。

            但为什么是以CSS这个话题作为开始呢？

            因为在web上，无障碍开发是构建在合理的HTML与CSS结构上的，

            当然也因为自己喜欢CSS，并且CSS在行业的地位没有JS高，也希望能调起同行对CSS的热情，所以就不自量力地开始了这个系列。

            一来是给自己一次系统学习的机会，二来也是想为自己后续的无障碍WEB设计的研究稳扎基础。当然这只是开始，也希望自己可以不忘初心，砥砺前行，从Hello一直到Deep。

            # 还会坚持吗？

            写文章是一场自我狂欢的行为，痛并快乐着。

            最开始写文章是源于掘金举办的 开发者写作计划 ，以此为契机开始了技术文的写作生涯。写文章其实比写代码累多了，而且自己写的文章，其实喜欢的人不多。看着寥寥可数的阅读量，偶尔也会失落，也会想放弃或者学别人写一些百度就可以CV的爆款，但每次想起自己写作时所学习到的知识、训练出来的耐心以及小部分朋友的认可与鼓励，自己又咬咬牙坚持了下去。

            哈哈哈，自嘲一声 网黑写手 ，然后继续自我陶醉。
